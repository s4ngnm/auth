Quy trình Log Via mới mua về:

1.Sau khi mua via vào : m.facebook.com hoặc mbasic.facebook.com để Login tài khoản tránh bị checkpoint rồi sau đó vào fb.com bình thường.
Sau khi login ngâm tối thiểu 1 ngày trước khi vào trình quản lý quảng cáo

2.Logout toàn bộ session thiết bị cũ rồi đợi 2h sau mới change pass via: https://www.facebook.com/settings?tab=security

3.Log Via ngâm sau 1-2h mới đổi pass Via.
Link 1661 đổi pass Via tránh Checkpoint: https://www.facebook.com/privacy/review/?review_id=573933453011661&source=unknown

4.Link change pass hotmail (vừa login vừa thêm/đổi mail khôi phục và change pass luôn:
https://account.live.com/password/Change?mkt=en-US&refd=account.microsoft.com&refp=/&fref=home.drawers.security.change-password

Không nên đổi mail khác dễ bị checkpoint nick.

5.Tắt/bật lại 2FA: https://accountscenter.facebook.com/password_and_security/two_factor

6.Check mail gần đây và mail ẩn

   6.1https://www.facebook.com/recent_emails/security/?hide_back_button=1&from_accounts_center=1

   6.2 https://accountscenter.facebook.com/personal_info

7.Vào trang cá nhân lấy thông tin via và lưu lại:
+ Tên via
+ Link trang cá nhân
+ ngày tháng năm sinh
- Tìm và đổi ảnh đại diện của via thành ảnh chân dung rõ mặt nhất, lưu ảnh về để làm phôi xmdt, có avatar chân dung rõ mặt rồi thì lưu ảnh lại thôi ko thay mới
- Vào link facebook.com/accountquality check chất lượng tài khoản của via
- Vào trình quản lý quảng cáo, bấm 'Đồng ý' với quy định của Facebook, sau đó quay trở về facebook ngâm

 

Code change mail cho Via Clone:

https://www.facebook.com/settings?tab=account&section=email&view
