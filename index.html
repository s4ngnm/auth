    Quy trình xử lý Via Clone/Limit:

    1. Mua Via đăng nhập:
    - mbasic.facebook.com
    - Trên trình duyệt đăng nhập thêm các web khác ( google, twitter, youtube,...).
    - Log fb check đúng email và không có email lạ + sđt lạ.
    - 5 KHÔNG (Like, cmt, share, đổi tên, add friend) 5 ngày.
    - 1 tuần (1 tháng thì ok )
    - Change pass mail hiện tại và add/change mail khôi phục.
    => fb.com/hacked để đổi thông tin.
    mbasic.facebook.com/hacked tích dòng cuối để change pass (tích dòng 2 để change pass và xóa các mail và sđt lạ)
    - Xóa sạch thiết bị, xóa hết lịch sử đăng nhập, vị trí đăng nhập cũ... => log out via để xóa session cookie, rồi đăng nhập lại bằng mbasic.facebook.com
    - sau 2h tắt 2fa rồi bật lại 2fa để lấy mã 2fa mới.
    2. Ngâm trình quản lý quảng cáo:
    - fb.com/pe để đó 2–3 ngày không làm gì.
    - Kich BM https://business.facebook.com/overview/
    3. Share tài khoản quảng cáo + tạo page:
    - Tạo Page đăng bài.
    - Share Via khác 2-3 Via cầm.
    - Xác định con Via lên camp chạy quảng cáo và share page 1-2 con.
    4. Add thẻ (Dùng via gốc add thẻ cho tài khoản quảng cáo) ngâm thêm 1 ngày rồi mới lên camp.
    5. Viết bài quảng cáo + Seeding cho bài đó + Seeding toàn page.
    6. Dùng con Via cầm page để set camp (Không chỉnh sửa hôm đầu nhưng trong lúc chưa duyệt thì sửa dc =))) )

    Kinh nghiệm nuôi Clone:
    Mới mua về log mbasic ngâm 1-2h sau (acc yếu hoặc để lâu ko đăng nhập thì ngâm lâu hơn 24h):
    Thêm avatar, ảnh nền.
    Thêm thông tin cho nick nếu chưa có (Ngày sinh, quê quán, trường học,…) cái này lên mấy trang Gen info mà lấy bộ info đầy đủ để sau này lên via hoàn chỉnh.
    Thêm mail bằng code F12 và verify mail nếu chưa có mail. Có rồi thì ko đổi mail nha.
    Quy trình tương tác nuôi clone:
    Ngày 1-5: Lướt newfeed, tương tác với list group, page có sẵn, post 1 bài/ngày.
    Ngày 5-7: Kết bạn theo UID, tham gia list group, tương tác group.
    Từ ngày 7: Đổi thông tin, kết bạn theo UID, tham gia group, tương tác group với clone spam thì không cần đổi IP, trước khi nuôi dàn nhiều tk (>100 con) thì reset wifi nếu xài IP nhà máy ngoài ko proxy.
